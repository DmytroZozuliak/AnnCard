# 🌷 AnnCard — привітання з Днем народження для Анюти

Мобільний SPA-сайт із покроковими компліментами та анімаціями.
Зроблено на **Vite + React + Framer Motion**.

## 🚀 Локальний запуск

```bash
npm install
npm run dev
```

Відкрий посилання, яке покаже Vite (зазвичай http://localhost:5173).
Найкраще виглядає на телефоні або у мобільному режимі браузера (DevTools → Toggle device toolbar).

## 📦 Збірка

```bash
npm run build      # збірка у папку dist/
npm run preview    # локальний перегляд зібраної версії
```

## 🌍 Деплой на GitHub Pages

> ⚠️ Перед деплоєм перевір, що `base` у [vite.config.js](vite.config.js) збігається
> з назвою репозиторію (`/AnnCard/`). Якщо назва репозиторію інша — зміни цей рядок.

### Варіант A — автоматично (GitHub Actions, рекомендовано)

1. Створи репозиторій `AnnCard` на GitHub і запуш код у гілку `main`.
2. У налаштуваннях репозиторію: **Settings → Pages → Build and deployment → Source: GitHub Actions**.
3. Кожен push у `main` автоматично збирає й публікує сайт
   (workflow: [.github/workflows/deploy.yml](.github/workflows/deploy.yml)).

Сайт буде доступний за адресою:
`https://<твій-нікнейм>.github.io/AnnCard/`

### Варіант B — вручну через npm

```bash
npm run deploy
```

Ця команда збере проєкт і запушить папку `dist/` у гілку `gh-pages`.
Потім у **Settings → Pages** обери джерело: гілка `gh-pages`.

## ✨ Кроки привітання

1. Стартовий екран — «Почати своє свято»
2. Яка ти прекрасна (квіти, що розквітають)
3. Яка ти добра (тепле серце з променями)
4. Як ти дбаєш про мене (вітамінки, чай, їжа)
5. Мій нічний вартовий (центр управління вночі)
6. Ти — мій дім (затишок і тепло)
7. Як мені пощастило бути з тобою (конюшина на щастя)
8. Як сильно я тебе люблю (фінал + конфеті)

Тексти легко змінити в масиві `STEPS` у файлі [src/App.jsx](src/App.jsx).
